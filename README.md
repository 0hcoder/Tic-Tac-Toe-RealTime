````markdown
This project transforms the classic game into a dynamic multiplayer experience, utilizing the power of Socket.IO and React.
````

### Prerequisites

* **Programming Language:** JavaScript
* **Package Manager:** Npm

### Installation

```bash

cd Tic-Tac-Toe-RealTime
npm install
```

### Usage

```bash
npm start
```

